## How to use?

On the website, log in with user = 40028922 and pass = admin

The database is always reset

You can create sectors, employees for these sectors, and questions for both.

There are 2 screens, the admin screen (when logging in with the above credentials), and the employee screen, which you can access by logging in with the credentials of an employee you created.

Employees can answer questions and view their response history
